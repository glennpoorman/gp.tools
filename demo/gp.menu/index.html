Simple Menu

A simple single layer menu optionally allowing a pulldown menu for each main menu item.

The HTML

The HTML here is extremely simple. Your top level menu is a simple HTML unordered list. In order to be recognized as a menu bar, the id of the list must be set to "gp-menu-bar" as in the following sample.
<ul id="gp-menu-bar">
Each list item within the menu bar element must contain, at a minimum, one anchor element which represents the main selection. So using the "One" selection above in this page as an example, the menu bar and first item would read:
<ul id="gp-menu-bar"> <li><a href="#">One</a></li>
Of course, you would put in your own destination in place of the "#". Now to have the main menu bar selection also bring up a submenu, you add a single div element that contains any number of additional selections. No added classes or ids are necessary.

So combining the first and second selections in the menu bar of this page, we get:
<ul id="gp-menu-bar"> <li><a href="#">One</a></li> <li><a href="#">Two</a> <div> <a href="#">Two-A</a> <a href="#">Two-B</a> </div> </li>
To get a clearer picture of the setup for a full menu, use your browser to view the full source of this page.

The JavaScript

There are no public functions to be called. In order to execute the JavaScript setup code, you only need to pull in the JS file gp.tools.js and the menu intialization code will run automatically on document load. There actually isn't much to the JavaScript code. On document load, the initialization code tags the menu item anchors with ids and sets event handling code for mouse enter and mouse exit so that the popups work properly.

Customization

The aesthetics of the menu are controlled entirely by styles. The included CSS file gp.tools.css contains defaults which are used on this page. For additional tweaking of things like fonts, colors, font sizes, etc, you can override any of the styles in the included CSS file to suit your needs.

For example, the most common customization would most likely be the colors. To modify the background and text color of the main menu items, you would want to override the background and color properties for the element ul#gp-menu-bar li a. You might also modify the color that displays when the mouse hovers over the menu item by overriding the background property for the element ul#gp-menu-bar li a:hover. To modify the same colors for the items in the submenu, you can modify the element ul#gp-menu-bar div.

Basically, any and all styles are customizable. If you have a pretty good handle on CSS, you can start with the provided file and go nuts!

Styles available for override are:
ul#gp-menu-bar - The top menu bar.
ul#gp-menu-bar li - Individual top menu bar items.
ul#gp-menu-bar li a - Anchors within the individual top menu bar items.
ul#gp-menu-bar li a:hover - Hover behavior in top menu bar anchors.
ul#gp-menu-bar div - Popup menus.
ul#gp-menu-bar div a - Anchors within the popup menus.
