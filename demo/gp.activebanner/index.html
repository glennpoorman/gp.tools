Responsive Navigation

Responsive banner and menu for compatibility with mobile devices.

The active banner expands on the Gp.Menu tools to make websites compatible with mobile devices. The banner is split into two halves, a title section and a navigation section. The developer can create any type of title (text based, an image, etc) and the navigation section contains the "gp-menu-bar" element from gp.menu.js.

To watch the responsive banner in action, you can change your browser to mobile display (if your browser supports this as a development tool) or you can simply shrink the width of your browser window so that it is less than 640 pixels.

The HTML

The HTML is still fairly simple but requires a little more care than the simple menu sample. For the banner to work, there must be two div elements defined. One is the title section and the other is the navigation section.
<div id="gp-title-section"> ... </div> <div id="gp-nav-section"> ... </div>
The title section can contain whatever you want to use as a title be it a text string or some other type of object such as an image. It can even be empty but the section must exist in order to serve as a place holder for the mobile menu icon. On this page we simply use the title gp.activebanner.
<div id="gp-title-section"> gp.activebanner </div>
The navigation section contains a single nav element and also contains the unordered list that will represent our menu bar. Note that the id gp-menu-bar is no longer required. The initialization code will take care of marking any other elements that require ids.
<div id="gp-nav-section"> <ul> <li><a href="#">One</a> <li><a href="#">Two</a> <div> <a href="#">Two-A</a> <a href="#">Two-B</a> </div> </li> </div>
A good practice then is to wrap both sections in some manner of container element. This isn't a necessity but allows for further customization (discussed below). On this page, the banner is enclosed in a header element. When you put it all together then, your full banner and menu looks as follows:
<header> <div id="gp-title-section"> gp.activebanner </div> <div id="gp-nav-section"> <ul> <li><a href="#">One</a> <li><a href="#">Two</a> <div> <a href="#">Two-A</a> <a href="#">Two-B</a> </div> </li> </div> </header>

The JavaScript

As with the menu tools, there are no public functions to be called. In order to execute the JavaScript setup code, you only need to pull in the JS file gp.tools.js and the banner initialization code will run automatically on document load. In addition to setting up the ids for the menu items and the mouse events, the initialization code for the banner also adds the mobile menu icon which defaults to being hidden but will appear on mobile devices.

Customization

Like the menu, the aesthetics are controlled entirely by the included CSS file. Again, best practice is to start with the styles provided and then override any individual styles you want to customize.

For example, the most common customization would most likely be the colors. To modify the background and text color of the main menu items, you would want to override the background and color properties for the element ul#gp-menu-bar li a. You might also modify the color that displays when the mouse hovers over the menu item by overriding the background property for the element ul#gp-menu-bar li a:hover. To modify the same colors for the items in the submenu, you can modify the element ul#gp-menu-bar div.

Basically, any and all styles are customizable. If you have a pretty good handle on CSS, you can start with the provided file and go nuts!

Styles available for override (in addition to the menu) are:
a#gp-menu-icon - The mobile menu icon.
div#gp-nav-section - The navigation section.
div#gp-title-section - The title section.
nav#gp-active-nav - The nav element containing the menu bar.
In addition, there is a media query set to kick in when the screen width goes below 640 pixels that tweaks the following styles:
a#gp-menu-icon
div#gp-nav-section
div#gp-title-section
nav#gp-active-nav
ul#gp-menu-bar li
ul#gp-menu-bar li a
ul#gp-menu-bar div
ul#gp-menu-bar div a
