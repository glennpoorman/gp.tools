Gp.AudioAlbum

HTML Audio Album

Utilities for displaying audio elements in a page using MediaElement

Like the SlideShow vs SlideDeck discussion, the audio player tools work great for scattering audio elements on a web page and having complete control over each elements position in relation to the other content on the page. Sometimes though, you want to essentially create an album on the page. A no frills container that holds all of the elements in a collection all grouped in one place. Similar to the SlideDeck tools, a strategically marked element in the page is all you need and the tools will handle the rest.

audio.json

All of the data for the audio elements must be put into a JSON file for the album tools to work. The format of the JSON file is exactly like the format in the gp.audioplayer discussion. One difference to note here though is that the previously unused "title" property in the collection JSON will be used here but only if specified. The property itself is still optional.

The HTML

With the album, you no longer need to create elemenets in your HTML for each audio file. For this to work, it is simply a matter of placing a single container element for the entire album.
<div id="gp-audio-album"></div>
When the "fromJSON" function is called, the code will automatically look for an element with the id "gp-audio-album" and automatically create the individual audio elements in order within that container. In addition, if a "title" is specified on the JSON collection, an additional DIV element will be created and the title will be inserted.

That's it!

Customization

In addition to the styles that can be used to customize the look of the individual audio elements, two style definitions can be used to customize the appearance of the album. The album itself must have the id "gp-audio-album". In addition, if a title is present the DIV element that will contain the title will automatically have the class "gp-audio-album-title" applied to it. By default, the style for the class "gp-audio-album-title" looks as follows:
.gp-audio-album-title { font-size : 1.5em; margin-bottom : 0.5em; }
That style can, of course, be further customized as can the style for the main "gp-audio-album" container. In this sample, we have customized the style as follows:
#gp-audio-album { background : #303060; border-radius : 8px; color : #CCC; padding : 1.0em; padding-bottom : 0.5em; width : 400px; }
