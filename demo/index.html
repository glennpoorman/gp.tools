GP.Tools (V2)

JavaScript Tools for Website Developers.

Written using JavaScript, jQuery, CSS, LightBox, and MediaElement

The Tools

The tools are encapsulated into a single JS file and a single CSS file. In addition, there are dependencies on the 3rd party tools jQuery, LightBox, and MediaElement. The proper versions of these tools are included in the distribution. The tools themselves are broken up into logical groups separated by functionality as follows:
  • Gp.Basic
    Some simple functions for inserting copyright, displaying single images, inserting email addresses, etc.

  • Gp.Menu, Gp.ActiveBanner
    Navigation tools for creating simple menus with single layer pulldowns and mobile friendly navigation bars.

  • Gp.SlideShow, Gp.SlideDeck
    Tools for displaying collections of images into a single web page grouped together for viewing using LightBox.

  • Gp.AudioPlayer, Gp.AudioAlbum
    Tools for inserting audio elements into your page and connecting them with local audio files.

Setup

Toolkit usage requires both a JavaScript (JS) file as well as a style sheet (CSS) file. In adition, jQuery is required for all functionality while some pieces also require LightBox (for image display) and MediaElement (for audio elements). The minimum set of files you would need are:
gp.tools.js
gp.tools.css
jquery-3.2.1.js
Depending on your usage, additionally required files are:
lightbox/js/lightbox-plus-jquery.js
mediaelement/mediaelement-and-player.js
Please note that while both LightBox and MediaElement also distribute required style sheet (CSS) files, these files are automatically loaded in with gp.tools.css. For the JS files, however, we leave pulling those in up to you.

Also note that this distribution delivers minimized versions of all of the above files.
jquery-3.2.1.min.js
lightbox/js/lightbox-plus-jquery.min.js
mediaelement/mediaelement-and-player.min.js
gp.tools.min.js
gp.tools.min.css

CSS

There are styles associated with many of the included tools some required and some optional and overridable. These styles can be pulled in via the gp.tools.css or gp.tools.min.css files. In addition, both the LightBox and MediaElement utilities ship with their own stylesheets. Both of those stylesheets are automatically pulled in with gp.tools.css though so there should never be a need to reference them directly.

To pull in all of the necessary styles then is simply a matter of including the one stylesheet in the <head> section of your page.
<link rel="stylesheet" href="gp.tools.css" />

JS

All of Gp.Tools functionality is encapsulated into the single gp.tools.js or gp.tools.min.js file. As mentioned before, jQuery is also required for all functionality so at a minimum, you need to pull in two files.
<script type="text/javascript" src="jquery-3.2.1.js"></script>
<script type="text/javascript" src="gp.tools.js"></script>
In addition, both LightBox and MediaElement are required for some functionality. In the cases where you need those tools as well, you will need to pull in those JS files before you include gp.tools.js. Worst case scenario is a page that uses both the imaging and the audio utilities which would look like:
<script type="text/javascript" src="jquery-3.2.1.js"></script>
<script type="text/javascript" src="lightbox/js/lightbox-plus-jquery.js"></script>
<script type="text/javascript" src="mediaelement/mediaelement-and-player.js"></script>
<script type="text/javascript" src="gp.tools.js"></script>
IMPORTANT: It's important to note that the Gp.Tools.js file has several tasks that run on document load and some of those tasks are dependent on the other utilities so the file gp.tools.js should always be pulled in last.
